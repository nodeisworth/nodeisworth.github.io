What is the difference between exports and module.exports ?

module.exports is the main object exported or returned from the file while exportsis a reference for module.exports so you can extend it by adding functions but not assigning it a new object I mean when using exports , you define a new function for example sum write this line : exports.sum=(x,y)=>x+y donot ever write this cause this will generate error exports = { sum: (x, y) => x + y } If you want to define a new object use module.exports module.exports={ sum:(x,y)=>x+y }
