Rabbitmq is based on erlang so we should install erlang first and then install rabbitmq

To install Erlang 

Windows:  Download erlang from here and run exe install

Ubuntu : sudo apt-get install erlang


After that install rabbitmq 

To install RabbitMQ 

Windows:  Download windows install from here

Ubuntu : sudo apt-get install rabbitmq
