Rabbitmq is a message broker but what is message broker?. Message broker is an application that allows for communication between multiple components using messages and it is an important factor as being the message transfer agent in the message oriented middle-ware architecture.

It uses the AMQP protocol which is an application layer protocol used in message oriented middle-ware and it define features including message queueing and routing and reliability and security


Who are the main actors used in rabbitmq :

Publisher : The service that send messages 

Subscriber : The service that receive messages 


How rabbitmq works:

Publisher push messages through queue called work queue and then subscriber fetch messages from this queue 


Also there are cases where publisher send message through middle ware called exchange which is then pushed to queue which is then pulled by subscriber
